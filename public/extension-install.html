Installation Steps

Step 1: Open Chrome Extensions

1. Open Google Chrome

2. Type chrome://extensions/ in the address bar and press Enter

3. Or click the three dots menu ‚Üí More tools ‚Üí Extensions

Step 2: Enable Developer Mode

1. In the top right corner, toggle on "Developer mode"

2. You should see new buttons appear: "Load unpacked", "Pack extension", etc.

Step 3: Load the Extension

1. Click "Load unpacked"

2. Navigate to your audit tool project folder

3. Select the browser-extension folder

4. Click "Select Folder"

Step 4: Verify Installation

1. You should see "Web Audit Screenshot Tool" in your extensions list

2. Make sure it's enabled (toggle should be blue)

3. Pin it to toolbar (click the puzzle piece icon, then the pin icon)

Step 5: Test the Extension

1. Go back to the audit tool page

2. Click "Check Again" in the Browser Extension section

3. You should see "Extension Detected" with capture options

Troubleshooting

Extension Not Detected?

1. Make sure the extension is enabled in chrome://extensions/

2. Refresh the audit tool page

3. Click "Check Again" button

4. Try reloading the extension (click üîÑ reload button in chrome://extensions/)

Capture Not Working?

1. Make sure both servers are running:

npm run dev:full

2. Check console logs by right-clicking extension icon ‚Üí "Inspect popup"

3. Verify API server is accessible at http://localhost:8081

Extension Folder Location

The extension files should be in your project at:

your-project-folder/browser-extension/

This folder should contain:

  • manifest.json
  • popup.html
  • background.js
  • content.js
  • icons/ folder
